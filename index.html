Score File Format

The score files for Addy's Adventures are stored in XML format. Each score file contains tags for instruments, notes, durations, and effects. Below is the structure of the XML format: 

<score bpm = "180" beatspermeasure = "4">
  <instrument instrument="AdditiveSynthesizer">
    <note measure="1" beat="1" duration="1.0" note="C4" harmonics="1.0,0.5,0.25" />
    <note measure="5" beat="1" duration="3.0" note="C4" harmonics="1.0,0.5,0.25" nextHarmonics="0.5,0.7,0.3" crossfadeDuration="3.0" />
  </instrument>
  <instrument instrument="ToneInstrument">
    <note measure="1" beat="1" duration="1.5" note="G4" />
  </instrument>
</score>

Each <instrument> represents an instrument line, and each <note> defines an event with its musical parameters.

Component Description — Additive Synthesizer

The Additive Synthesizer component generates sound by combining multiple sine wave oscillators corresponding to harmonic overtones of a base frequency. Each oscillator can have independent amplitude and phase settings, allowing rich and dynamic tone creation.

Functionality

  • Generates harmonically related sine wave components
  • Supports adjustable harmonic amplitudes
  • Implements envelope shaping (ADSR)
  • Allows per-note parameter control through the score XML

Ownership

Owned and implemented by Daniel Chen.

Supported Grading Elements

  • Sinusoid playback on demand from the sequencer
  • Definition of all harmonics
  • Envelope generation
  • Polyphony
  • Sound cross-fading
